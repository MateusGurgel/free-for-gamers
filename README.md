## Run Locally

Clone the project

```bash
  git clone https://github.com/MateusGurgel/free-for-gamers
```

Go to the project directory

```bash
  cd free-for-gamers
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open your browser to see the result!


## Tech Stack

**Client:** React, NEXT.JS, SWR, TailwindCSS, Framer-motion, chakra.UI
